## 3. 資金流向圖

```
用戶（MetaMask）
      │
      │ approve() + subscribe(100 USDC)
      ▼
┌─────────────────────────────────────┐
│         RWAFund.sol                  │
│                                      │
│  費用 = 100 × 0.5% = 0.5 USDC      │
│  淨額 = 99.5 USDC                   │
│                                      │
│  代幣 = 99.5 × 1e18 / navPerToken   │
│        = 99.5 × 1e18 / 100e6        │
│        = 0.995 M7F                   │
│                                      │
│  ┌─────────────────┐                 │
│  │ USDC 池        │                 │
│  │ (合約持有)     │                 │
│  └─────────────────┘                 │
│                                      │
│  鑄造 0.995 M7F ──▶ 用戶地址        │
│  轉移 0.5 USDC  ──▶ feeCollector    │
└─────────────────────────────────────┘

用戶贖回：
      │
      │ redeem(0.995 M7F)
      ▼
┌─────────────────────────────────────┐
│  銷毀 0.995 M7F                     │
│  USDC總值 = 0.995 × navPerToken     │
│  費用 = 總值 × 0.5%                 │
│  淨額 ──▶ 用戶                      │
└─────────────────────────────────────┘
```

## 4. Black-Litterman 數學流程

```
市場數據（9個月）
        │
        ▼
┌─────────────────────────────────────────┐
│  Step 1: 協方差矩陣 Σ (7×7)             │
│  方法：Ledoit-Wolf 縮減                  │
└──────────────────┬──────────────────────┘
                   │
        ┌──────────┴──────────┐
        ▼                     ▼
┌───────────────┐    ┌────────────────────┐
│  市值加權     │    │  LLM 觀點          │
│  w_market     │    │  P (7×7 one-hot)   │
│  (7×1向量)   │    │  Q (7×1 報酬%)    │
└──────┬────────┘    │  Ω (7×7 對角矩陣) │
       │             └────────┬───────────┘
       ▼                      │
┌───────────────┐             │
│  均衡報酬     │             │
│  Π = δΣw     │             │
│  δ = 1.8     │             │
└──────┬────────┘             │
       │                      │
       └──────────┬───────────┘
                  ▼
┌─────────────────────────────────────────┐
│  BL 後驗報酬 E[R]                       │
│  = Π + τΣP'[PτΣP' + Ω]⁻¹(Q - PΠ)    │
│  τ = 0.15                               │
└──────────────────┬──────────────────────┘
                   │
                   ▼
┌─────────────────────────────────────────┐
│  均值-變異數最適化                       │
│  max w'E[R] - (δ/2)w'Σ_post w          │
│  s.t. Σwᵢ = 1                          │
│       0.05 ≤ wᵢ ≤ 0.30                 │
│  演算法：SLSQP                          │
└──────────────────┬──────────────────────┘
                   │
                   ▼
           最終權重 (bps)
```
